Pengenalan CSS

CSS adalah singkatan dari Cascading Style Sheets. CSS digunakan untuk mengontrol tata letak dan tampilan halaman web.

Selektor CSS

Selektor digunakan untuk "memilih" elemen HTML yang ingin Anda gaya. Berikut adalah beberapa selektor dasar:

  • p - Memilih semua elemen paragraf
  • .classname - Memilih semua elemen dengan kelas tertentu
  • #idname - Memilih elemen dengan ID tertentu

Contoh:


/* Memilih semua paragraf dan mengatur warnanya menjadi biru */
p {
    color: blue;
}

/* Memilih elemen dengan kelas 'highlight' dan mengatur background-nya */
.highlight {
    background-color: yellow;
}

/* Memilih elemen dengan ID 'header' dan mengatur ukurannya */
#header {
    font-size: 24px;
}

Properti CSS

Properti CSS digunakan untuk mengatur gaya elemen. Berikut adalah beberapa properti umum:

  • color - Mengatur warna teks
  • background-color - Mengatur warna latar belakang
  • font-size - Mengatur ukuran font
  • margin - Mengatur margin luar elemen
  • padding - Mengatur padding dalam elemen

Contoh:


p {
    color: green;
    background-color: lightgray;
    font-size: 18px;
    margin: 10px;
    padding: 5px;
}

Tata Letak CSS

Display

Properti display menentukan bagaimana elemen ditampilkan. Nilai umum termasuk block, inline, dan inline-block.

Contoh:


.block-element {
    display: block;
}

.inline-element {
    display: inline;
}

.inline-block-element {
    display: inline-block;
}

Flexbox

Flexbox adalah model tata letak yang kuat untuk mendesain tata letak responsif. Flexbox menggunakan properti seperti display: flex, justify-content, dan align-items.

Contoh:


.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

.item {
    margin: 10px;
}

Desain Responsif

Desain responsif memastikan bahwa halaman web terlihat baik pada semua perangkat. Media queries digunakan untuk menerapkan gaya yang berbeda pada ukuran layar yang berbeda.

Contoh:


/* Gaya untuk perangkat dengan lebar maksimum 600px */
@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

/* Gaya untuk perangkat dengan lebar minimum 601px */
@media (min-width: 601px) {
    body {
        background-color: lightgreen;
    }
}
