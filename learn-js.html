Pengenalan JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web interaktif. JavaScript dapat digunakan untuk memperbarui dan mengubah baik HTML maupun CSS.

Variabel dan Tipe Data

Variabel dalam JavaScript dapat dideklarasikan menggunakan var, let, atau const. Berikut adalah beberapa tipe data dasar dalam JavaScript:

  • String - Contoh: "Hello, World!"
  • Number - Contoh: 42
  • Boolean - Contoh: true atau false
  • Array - Contoh: [1, 2, 3]
  • Object - Contoh: { name: "John", age: 30 }

Contoh deklarasi variabel:


let name = "John";
const age = 30;
var isStudent = false;

Fungsi

Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu. Fungsi didefinisikan dengan kata kunci function diikuti oleh nama fungsi, tanda kurung, dan kurung kurawal:

Contoh:


function greet(name) {
    return "Hello, " + name + "!";
}

console.log(greet("Alice")); // Output: Hello, Alice!

Kondisi

Kondisi digunakan untuk melakukan tindakan yang berbeda berdasarkan kondisi yang berbeda. Pernyataan if digunakan untuk mengevaluasi kondisi:

Contoh:


let score = 85;

if (score >= 90) {
    console.log("Grade: A");
} else if (score >= 80) {
    console.log("Grade: B");
} else {
    console.log("Grade: C");
}

Loop

Loop digunakan untuk menjalankan blok kode berulang kali. Berikut adalah beberapa jenis loop yang umum dalam JavaScript:

For Loop

Contoh:


for (let i = 0; i < 5; i++) {
    console.log("Number: " + i);
}

While Loop

Contoh:


let i = 0;

while (i < 5) {
    console.log("Number: " + i);
    i++;
}
