Pengenalan HTML

HTML adalah singkatan dari HyperText Markup Language. Ini adalah bahasa markup standar untuk membuat halaman web. Elemen HTML adalah blok penyusun dari halaman HTML.

Dasar-dasar HTML

Format Teks

HTML menyediakan beberapa elemen untuk memformat teks. Berikut adalah beberapa yang paling umum:

  • <b> - Teks tebal
  • <i> - Teks miring
  • <u> - Teks bergaris bawah
  • <mark> - Teks yang disorot

Tabel HTML

Tabel didefinisikan menggunakan elemen <table>. Tabel dibagi menjadi baris (<tr>), dan setiap baris dibagi menjadi sel (<td> atau <th> untuk sel header):

Contoh:

Header 1 Header 2
Data 1 Data 2

Daftar HTML

Daftar Berurutan

Daftar berurutan dibuat menggunakan elemen <ol>:

Contoh:

  1. Item pertama
  2. Item kedua
  3. Item ketiga

Daftar Tak Berurutan

Daftar tak berurutan dibuat menggunakan elemen <ul>:

Contoh:

  • Item pertama
  • Item kedua
  • Item ketiga
